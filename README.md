# Langton-s-Ant

Langton's ant is one of the most recognized cellular automata (turmites). Turmites can produce complex patterns (spirals, golden rectangles, chaotic structures, nested growth and circular structures). In this case, where the cells are square, in the first moves it makes it creates simple patterns which are most often symmetrical. After a few hundred steps a chaotic structure prevails with the ant forming supposedly random patterns. However, when it approaches 10,000 steps the ant enters a loop and forms an “avenue”, as shown in the figure below.
